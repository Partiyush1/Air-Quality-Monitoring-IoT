Air Quality Monitoring System – IoT Project

This project monitors real-time Air Quality, Temperature, and Humidity using ESP32, MQ135, and DHT11 sensors. Data is uploaded to ThingSpeak Cloud and displayed on a custom website dashboard with graphs and alerts.

Features

Real-time air quality monitoring

Temperature & humidity tracking

Cloud uploading using ThingSpeak

Live dashboard using JavaScript

Air Quality classification (Good/Moderate/Bad)

Buzzer alert on dangerous AQI

Fully responsive website UI

Technologies Used

ESP32 Microcontroller

MQ135 Gas Sensor

DHT11 Sensor

ThingSpeak Cloud

HTML, CSS, JavaScript

Wokwi Simulator
